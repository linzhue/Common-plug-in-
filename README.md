# Common-plug-in-
# 公共组件结构

```bash

	├── basetree # 基础树形组件
	├── deptree # 部门人员树形组件
	├── dialog # 弹出框组件
	├── grid # 分页表格组件
	├── pagination # 分页组件
	├── print # 打印组件
	├── organCheck # 机构代码名称校验组件
	├── organSearch # 机构代码名称查询组件
	├── search # 查询组件
	├── tabroll # 滚动Tab组件
	├── textSearch # 文字查询高亮插件
	├── tooltip # 工具提示
	├── unitselect # 部门选择组件
	├── unitselect-mpage # 部门分类多页选择组件
	├── unitselect-spage # 部门分类单页选择组件
	├── uploader # 上传下载组件
```
