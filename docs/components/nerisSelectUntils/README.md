# 部门分类组件
该类组件主要用来选择机构，支持多选和单选、查询、删除、清空和数据反显等功能。

- [部门分类选择组件](nerisSelectUnits.md)
- [部门分类单页选择组件](nerisUnitsSinglePage.md)
- [部门分类多页选择组件](nerisUnitsMultiPage.md)

## 依赖
```html
jQuery
bootstrap
NerisUI
```
