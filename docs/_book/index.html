注意： 集成组件 1.26-SNAPSHOT 版本后需要在你项目的 spring-mvc.xml 文件中增加一行对 org.csits 包的扫描配置：

<context:component-scan base-package="org.csits"/>

组件当前版本依赖：

  • 权限接口版本 urmsInterface.version : 1.8.13-RELEASE
  • 非结构化版本 oltp-file-service.version : 1.2.3-RELEASE

1. 组件坐标

<dependency>
    <groupId>org.neris</groupId>
    <artifactId>neris-tech-widget-webapp</artifactId>
    <version>1.26-RELEASE</version>
    <type>war</type>
</dependency>

2. 非结构化坐标

<dependency>
  <groupId>org.neris</groupId>
  <artifactId>oltp-file-service</artifactId>
  <version>1.2.3-RELEASE`</version>
</dependency>

3. 版本变更说明

  • 新增分页表格组件，实现了表格排序功能。
